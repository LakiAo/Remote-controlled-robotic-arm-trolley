# 一款参加学校比赛所用的智能车代码
## Remote-controlled-robotic-arm-trolley
The car is made for STAS-Science and technology exhibition.  
参考了学长的代码并添加了循迹以及多任务处理（确保遥控和彩灯能同时运行）的代码  
期间也学习到了不少东西
